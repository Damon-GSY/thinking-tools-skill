# Thinking Tools Skill

基于万维钢《现代思维工具词典》的 OpenClaw AgentSkill。

## 概览

- **148 个思维工具**，按 7 大类别 + 6 大基本世界观分层组织
- 自动激活：当用户需要分析问题、做决策、评估方案时
- 每个工具包含：来源、释义、应用、洞见

## 层级结构

```
六大基本世界观
├── 认知偏差 (10)
├── 概率与决策 (15)
├── 系统思维 (47)
├── 战略与管理 (22)
├── 创新与创造力 (12)
├── 复杂网络与信息论 (11)
└── 工程与科学方法 (31)
```

## 安装

```bash
# 克隆到 OpenClaw workspace skills 目录
cp -r thinking-tools/ ~/.openclaw/workspace/skills/
```
